Eneido SO (OS) es un kernel hecho desde 0 en c++

Tiene las siguientes caracteristicas:                             
                                                              
  Driver VGA                                                                      
  VFS (Sistema de archivo virtual)                                      
  GDT (Tabla de descriptores global)                                             
  Shell (Terminal de comandos)                                    
  GameHub (Interfaz de juegos)                        
  Simple Math (hasta las funciones seno y coseno)                
  GRUB (Bootloader)                                
                                                  
Cosas por agregar:                            
                                        
  IDT (Tabla de descriptores de interrupciones)            
  ATA - Storage Driver (que funcione)        
  Raycasting (Juego)                        
  Timer                                        
  Random generator
