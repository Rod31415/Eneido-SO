# Features

✔️ 640 x 480 window.  
✔️ DOEN Interpreter( like BASIC )  
✔️ Interrupts  
✔️ Virtual File System  
✔️ RTC (Real Time Clock) Driver  
✔️ PCI Driver  
✔️ Basic memory & TTY functions  
✔️ Global Descriptor Table  
✔️ File editor built in  
✔️ Snake game  
✔️ RTL8139 Driver  
✔️ Keyboard Driver  
✔️ IRQs  
✔️ ELF Loader
